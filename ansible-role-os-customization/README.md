# Ansible Role: os-customization
It applies custom settings for each virtual machine after it was deployed

## Requirements
This role is valid for any Linux distribution and Windows systems

## Dependencies
None

## Example playbook:
    - hosts:
      roles:
        - canvia.os-customization
