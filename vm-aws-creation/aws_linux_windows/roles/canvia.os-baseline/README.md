# Ansible Role: os-baseline
It configures a standard OS baseline according to our internal best practices

## Requirements
This role is valid for any Linux distribution and Windows systems

## Dependencies
None

## Example playbook:
    - hosts:
      roles:
        - canvia.os-baseline
